# CrewAI Video Blog Writer

### How to run this app

1. Creaing Environment
```bash
conda create -p venv python=3.10 -y

source activate venv/
```

2. Installing Required Packages
```bash
pip install -r requirements.txt

```
3. To Run the app
```bash
python crew.py

```
